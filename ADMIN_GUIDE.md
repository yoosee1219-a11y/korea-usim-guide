# 관리자 페이지 사용 가이드

## 🎉 완성된 기능

### ✅ 블로그 관리 시스템
- 글 작성/수정/삭제
- WYSIWYG 에디터 (React Quill)
- 이미지 삽입
- 카테고리/태그 관리
- 임시저장/발행 기능
- 미리보기
- URL 슬러그 자동 생성

---

## 🔐 접속 방법

### 1. 관리자 페이지 URL
```
https://koreausimguide.com/admin/blog
```

### 2. 로그인
- **기본 비밀번호:** `admin123`
- ⚠️ **중요:** 배포 전에 반드시 비밀번호를 변경하세요!

**비밀번호 변경 방법:**
1. `.env.local` 파일에 추가:
   ```
   ADMIN_PASSWORD=your-secure-password
   ```
2. `app/admin/blog/page.tsx` 파일에서 `admin123` 부분 수정

---

## 📝 블로그 글 작성하기

### 1단계: 새 글 작성 페이지 이동
1. `/admin/blog` 접속
2. "새 글 작성" 버튼 클릭

### 2단계: 글 작성
1. **제목** 입력 (필수)
2. **URL 슬러그** 자동 생성 (수정 가능)
3. **카테고리** 선택:
   - 가이드
   - 비교
   - 뉴스
   - 팁
   - 리뷰
4. **태그** 추가 (Enter로 추가)
5. **대표 이미지 URL** 입력 (선택사항)
6. **요약** 입력 (비워두면 자동 생성)
7. **본문** 작성 (WYSIWYG 에디터)

### 3단계: 저장/발행
- **임시저장:** 작성 중인 글 저장 (발행 안 됨)
- **발행하기:** 즉시 사이트에 공개

---

## 🖼️ 이미지 삽입 방법

### 방법 1: URL로 삽입
1. 에디터 도구바에서 이미지 아이콘 클릭
2. 이미지 URL 입력
3. 확인

### 방법 2: 대표 이미지
1. "대표 이미지 URL" 필드에 URL 입력
2. 미리보기 자동 표시

**이미지 호스팅 옵션:**
- Supabase Storage (권장)
- Cloudinary
- 직접 업로드 후 public 폴더

---

## ✏️ 글 수정하기

1. `/admin/blog` 목록에서 "수정" 클릭
2. 내용 수정
3. "임시저장" 또는 "발행하기" 클릭

---

## 🗑️ 글 삭제하기

1. `/admin/blog` 목록에서 "삭제" 클릭
2. 확인 팝업에서 "확인"

⚠️ **주의:** 삭제된 글은 복구할 수 없습니다!

## 📊 SEO 최적화 팁

### 1. 제목 작성 (Title)
- 50-60자 이내
- 핵심 키워드 포함
- 매력적이고 구체적으로

**예시:**
- ❌ "한국 유심 가이드"
- ✅ "한국 여행자를 위한 유심 구매 완벽 가이드 (2025)"

### 2. URL 슬러그 (Slug)
- 소문자 사용
- 하이픈(-)으로 단어 구분
- 핵심 키워드 포함

**예시:**
- ❌ "guide_1234"
- ✅ "korea-usim-guide-for-travelers-2025"

### 3. 요약 (Excerpt)
- 150-160자 이내
- 글의 핵심 내용 요약
- 클릭을 유도하는 문구

### 4. 카테고리 & 태그
- 카테고리: 3-5개 이내로 일관성 유지
- 태그: 5-10개, 구체적인 키워드

### 5. 이미지 최적화
- Alt 텍스트 추가 (SEO)
- 적절한 크기 (너비 1200px 권장)
- WebP 형식 권장

### 6. 내부 링크
- 관련 글로 링크 추가
- "여기" 대신 구체적인 앵커 텍스트 사용

**예시:**
- ❌ "자세한 내용은 [여기](링크)를 참고하세요."
- ✅ "[SKT vs KT 요금제 비교](링크)를 확인하세요."

## 🔧 문제 해결

### 문제: 로그인이 안 돼요
**해결:**
1. 비밀번호 확인 (기본: `admin123`)
2. 브라우저 쿠키/캐시 삭제
3. 시크릿 모드로 시도

### 문제: 글이 저장 안 돼요
**해결:**
1. 제목과 내용이 입력되었는지 확인
2. 브라우저 콘솔 (F12) 에서 에러 확인
3. Supabase 연결 상태 확인

### 문제: 이미지가 안 보여요
**해결:**
1. 이미지 URL이 올바른지 확인
2. 이미지 URL이 HTTPS인지 확인
3. CORS 설정 확인

### 문제: 번역이 안 돼요
**해결:**
1. `.env.local`에 Supabase 키 확인
2. 인터넷 연결 확인
3. Google Translate API 사용량 확인
