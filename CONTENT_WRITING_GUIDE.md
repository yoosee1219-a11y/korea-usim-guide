# 📝 KOREAUSIMGUIDE 컨텐츠 작성 가이드라인

> 클로드 AI에게 전달할 컨텐츠 작성 지침서
>
> **목표**: 리치 스니펫에 잘 걸리고, 가독성이 뛰어난 SEO 최적화 컨텐츠 작성

---

## 🎯 핵심 원칙

1. **리치 스니펫 최적화**: JSON-LD 스키마 자동 생성 지원
2. **가독성 우선**: 스타일러 형식의 깔끔한 레이아웃
3. **SEO 친화적**: 구조화된 데이터와 명확한 헤딩 구조
4. **사용자 경험**: 목차, FAQ, 단계별 가이드 제공

---

## 📋 마크다운 작성 형식

### 1. 제목 구조 (H2, H3, H4)

**자동 기능:**

- H2 (`##`) → 빨간 배경, 흰색 텍스트 (스타일러 형식)
- H3 (`###`) → 일반 제목
- H4 (`####`) → 소제목
- **목차(TOC) 자동 생성**: 모든 헤딩이 목차에 자동 추가됨

**작성 예시:**

```markdown
## 1. 유심 수령이 중요한 이유

### 1.1 한국 도착 후 가장 먼저 필요한 것

#### 인터넷 연결의 중요성
```

**주의사항:**

- H2는 주요 섹션 구분용 (자동으로 목차에 추가됨)
- H3, H4는 세부 내용 구분용
- 헤딩에 ID가 자동으로 생성되어 앵커 링크 지원

---

### 2. FAQ 작성 형식

**자동 기능:**

- FAQPage JSON-LD 스키마 자동 생성
- Google 검색 결과에 FAQ 리치 스니펫 표시

**작성 형식:**

```markdown
## 자주 묻는 질문(FAQ)

Q: 인천공항에서 유심을 어떻게 수령하나요?
A: 인천공항 1층 입국장에 위치한 통신사 로밍 센터(SKT, KT, LG U+)에서 유심을 수령할 수 있습니다. 온라인으로 예약했다면 바우처(QR코드)를 제시하면 빠르게 수령할 수 있습니다.

Q: 유심 수령 전 어떤 준비가 필요한가요?
A: 온라인 예약 시 받은 바우처(QR코드)를 오프라인에서도 볼 수 있도록 캡처하여 저장하는 것이 좋습니다. 공항 Wi-Fi가 불안정할 수 있기 때문입니다.

Q: 통신사 카운터는 24시간 운영되나요?
A: SKT, KT, LG U+ 등의 메이저 통신사 카운터는 대부분 24시간 운영되지만, 알뜰폰 및 여행사 부스는 운영 시간이 제한될 수 있으므로 도착 시간을 미리 확인해야 합니다.
```

**패턴:**

- `Q:` 또는 `Q.`로 질문 시작
- `A:` 또는 `A.`로 답변 시작
- 각 FAQ는 빈 줄로 구분 (선택사항)

**자동 처리:**

- FAQ가 자동으로 추출되어 FAQPage 스키마 생성
- 리치 스니펫에 FAQ 표시

---

### 3. 단계별 가이드 (HowTo 스키마)

**자동 기능:**

- HowTo JSON-LD 스키마 자동 생성
- Google 검색 결과에 단계별 가이드 표시

**작성 형식:**

```markdown
## 인천공항 유심 수령 완벽 가이드

### 1. 예약 확인서 준비하기

온라인으로 미리 예약했다면, 이메일이나 앱으로 받은 바우처(QR코드)를 미리 캡처해 두세요.

### 2. 수령 장소(카운터) 찾기

대부분의 통신사 로밍 센터는 1층 입국장에 위치해 있습니다.

### 3. 현장 개통 및 테스트

유심을 받았다면 그 자리에서 바로 갈아 끼우고 데이터가 터지는지 확인하세요.
```

**패턴:**

- H2 또는 H3로 단계 제목 작성
- 각 단계는 명확한 제목과 설명 포함
- 번호 목록(`1.`, `2.`, `3.`)도 자동 감지

---

### 4. 일반 텍스트 작성

**스타일링:**

- 문단: 자동으로 적절한 간격과 줄 간격 적용
- **굵게**: `**텍스트**` → 자동 스타일링
- _기울임_: `*텍스트*` → 자동 스타일링
- 링크: `[텍스트](URL)` → 파란색, 호버 효과

**예시:**

```markdown
한국 도착 직후 가장 먼저 해야 할 일 중 하나는 통신 연결입니다. **인천공항에서 유심을 수령하는 방법**은 생각보다 간단하지만, 미리 알고 가면 훨씬 시간을 절약할 수 있습니다.

[공항 로밍 센터 위치 확인하기](https://example.com)
```

## 🎨 스타일러 형식 적용

### 제목 스타일링

**H2 (주요 섹션):**

- 자동으로 빨간 배경 + 흰색 텍스트
- 예: `## 1. 유심 수령이 중요한 이유`

**H3 (세부 섹션):**

- 일반 제목 스타일
- 예: `### 1.1 한국 도착 후 가장 먼저 필요한 것`

### 목차 자동 생성

**자동 처리:**

- 모든 H2, H3, H4가 자동으로 목차에 추가됨
- 클릭 시 해당 섹션으로 스크롤
- 스타일링된 목차 박스 자동 표시

**작성 시 고려사항:**

- H2는 주요 섹션 구분용으로 사용
- 명확하고 검색 친화적인 제목 작성

## 📊 JSON-LD 스키마 자동 생성

### 자동 생성되는 스키마

1. **Article 스키마**

   - 제목, 설명, 이미지, 날짜 자동 포함
   - 작성 불필요 (자동 생성)

2. **BreadcrumbList 스키마**

   - 홈 > 한국 통신 꿀팁 > [제목]
   - 작성 불필요 (자동 생성)

3. **HowTo 스키마**

   - H2/H3 헤딩과 번호 목록 자동 감지
   - 단계별 가이드 자동 생성

4. **FAQPage 스키마**
   - Q: A: 패턴 자동 감지
   - FAQ 자동 추출 및 스키마 생성

### 추가 스키마 (선택사항)

필요 시 마크다운에 직접 추가 가능:

```html
<script type="application/ld+json">
  {
    "@context": "https://schema.org",
    "@type": "Product",
    "name": "한국 유심 카드",
    ...
  }
</script>
```

## ✅ 컨텐츠 작성 체크리스트

### 필수 요소

- [ ] **제목 (H1)**: 명확하고 검색 친화적인 제목
- [ ] **요약 (excerpt)**: 150-200자 요약 (메타 설명용)
- [ ] **목차**: H2 헤딩으로 주요 섹션 구분 (최소 4-6개)
- [ ] **본문**: 상세하고 유용한 정보 제공 (2000-3000자)
- [ ] **FAQ**: 최소 5개의 FAQ 포함 (Q: A: 형식)
- [ ] **이미지**: 썸네일 이미지 URL (선택사항)

### 권장 요소

- [ ] **단계별 가이드**: HowTo 스키마를 위한 단계별 설명
- [ ] **관련 키워드**: 자연스럽게 키워드 포함
- [ ] **내부 링크**: 다른 꿀팁이나 요금제 페이지로 링크
- [ ] **실용적 팁**: Pro Tip, 주의사항 등 강조

## 📝 완성 예시

```markdown
# 인천공항 유심 수령 완벽 가이드

한국에 도착하자마자 가장 먼저 해야 할 일은 바로 통신 연결입니다. 인천공항에서 유심(USIM)을 수령하는 방법은 생각보다 간단하지만, 미리 알고 가면 훨씬 시간을 절약할 수 있습니다.

## 1. 유심 수령이 중요한 이유

한국 도착 후 가장 먼저 필요한 것은 인터넷 연결입니다.

유심을 교체하면 바로 통화와 데이터를 사용할 수 있어 편리합니다.

공항 와이파이는 끊김이 있을 수 있어 유심 사용이 더 안정적입니다.

## 2. 사전 예약 및 바우처 준비

온라인에서 미리 유심을 예약하면 공항 수령이 간편합니다.

예약 시 받은 바우처나 QR코드는 오프라인 저장이 필수입니다.

이메일로 받은 확인서를 캡처해두면 공항에서 쉽게 제시할 수 있습니다.

## 3. 수령 장소와 운영 시간

인천공항 유심 수령 부스는 대부분 입국장 1층에 위치합니다.

SKT, KT, LG U+의 로밍 센터는 24시간 운영되는 곳이 많습니다.

알뜰폰, 여행사 제휴 부스는 운영 시간이 정해져 있으니 확인 필수입니다.

## 4. 현장 수령 및 개통 절차

카운터에서 예약자 명과 바우처를 보여주면 바로 유심을 수령합니다.

직원이 직접 개통을 도와주는 경우가 많아 초보자도 걱정 없습니다.

유심을 교체한 뒤 데이터가 작동하는지 테스트해야 합니다.

## 5. 알아두면 좋은 팁

새벽 도착 시 편의점 유심 구매가 유용할 수 있습니다.

편의점 유심은 직접 개통 사이트에 접속해 등록해야 합니다.

데이터 중심 유심인지 음성 통화 포함 유심인지 확인하세요.

## 6. 자주 묻는 질문(FAQ)

Q: 유심은 한국 어디서나 사용할 수 있나요?
A: 네, 전국 어디서든 데이터와 통화가 가능합니다.

Q: 여권이 꼭 필요하나요?
A: 예, 신분 확인을 위해 여권이 필수입니다.

Q: 유심을 분실하면 어떻게 하나요?
A: 구입처에 문의하면 재발급 또는 대체 안내를 받을 수 있습니다.

Q: 편의점에서 산 유심도 외국인 개통이 가능한가요?
A: 네, 대부분 온라인 등록을 통해 개통 가능합니다.

Q: eSIM은 지원하나요?
A: 일부 통신사 부스에서는 eSIM도 지원하고 있습니다.
```

## 🎯 클로드에게 전달할 지시사항

### 입력 정보

1. **기존 상위 노출 글**: 참고할 기존 컨텐츠
2. **주제**: 작성할 컨텐츠 주제
3. **타겟 키워드**: SEO를 위한 주요 키워드

### 출력 요구사항

1. **마크다운 형식**으로 작성
2. **H2 헤딩**으로 주요 섹션 구분 (최소 4-6개)
3. **FAQ 섹션** 포함 (Q: A: 형식, 최소 5개)
4. **단계별 가이드** 포함 (HowTo 스키마용)
5. **150-200자 요약** 포함 (excerpt)
6. **검색 친화적 제목** 작성
7. **2000-3000자 분량**

### 스타일 가이드

- **톤**: 전문적이지만 친근한 톤
- **길이**: 2000-3000자 권장
- **구조**: 명확한 섹션 구분
- **키워드**: 자연스럽게 포함 (키워드 스터핑 금지)

## 📌 중요 참고사항

### 자동 처리되는 것들

✅ **목차**: H2, H3, H4 자동 감지 및 생성  
✅ **FAQ 스키마**: Q: A: 패턴 자동 추출  
✅ **HowTo 스키마**: 헤딩과 단계 자동 감지  
✅ **스타일링**: H2 빨간 배경, 링크, 텍스트 자동 스타일링  
✅ **Article 스키마**: 제목, 설명, 날짜 자동 생성

### 수동으로 작성해야 하는 것

❌ **제목 (H1)**: 명확하고 검색 친화적으로  
❌ **요약 (excerpt)**: 150-200자  
❌ **본문 내용**: 상세하고 유용한 정보  
❌ **FAQ 질문/답변**: Q: A: 형식으로 작성  
❌ **이미지 URL**: 썸네일 이미지 (선택사항)
