# Growth/Marketing: SEO & Growth Hacking

성장 마케팅 및 SEO 최적화

## 기능:

1. **SEO 최적화**
   - 키워드 리서치
   - 메타 태그 최적화
   - 구조화된 데이터 (Schema.org)
   - 내부 링크 최적화
   - 사이트맵 자동 생성
   - 로봇 크롤링 최적화

2. **성능 최적화 (Core Web Vitals)**
   - LCP (Largest Contentful Paint) < 2.5s
   - FID (First Input Delay) < 100ms
   - CLS (Cumulative Layout Shift) < 0.1
   - 이미지 최적화 (WebP, AVIF)
   - 코드 스플리팅
   - CDN 활용

3. **A/B 테스팅**
   - 랜딩 페이지 테스트
   - CTA 버튼 최적화
   - 헤드라인 테스트
   - 가격 실험
   - 통계적 유의성 검증

4. **성장 해킹**
   - 바이럴 루프 설계
   - 추천 프로그램
   - 온보딩 최적화
   - 리텐션 개선
   - 재활성화 캠페인

## SEO 체크리스트:

**On-Page SEO:**
- [ ] Title 태그 최적화 (50-60자)
- [ ] Meta Description (150-160자)
- [ ] H1-H6 계층 구조
- [ ] 이미지 Alt 텍스트
- [ ] URL 구조 최적화
- [ ] 내부 링크 전략

**Technical SEO:**
- [ ] Sitemap.xml 생성
- [ ] Robots.txt 설정
- [ ] 구조화된 데이터
- [ ] 모바일 친화적
- [ ] 페이지 속도 최적화
- [ ] HTTPS 사용

**Content SEO:**
- [ ] 키워드 밀도 (1-2%)
- [ ] LSI 키워드 포함
- [ ] 콘텐츠 길이 (최소 1000자)
- [ ] 정기적 업데이트
- [ ] 멀티미디어 포함

## 실험 결과:

```markdown
# A/B Test: 헤드라인 변경

**변형 A (기존):** "한국 유심 요금제 비교"
**변형 B (신규):** "한국 최저가 유심 찾기 - 1분 비교"

결과:
- 클릭률: 2.3% → 3.8% (+65%)
- 전환율: 1.2% → 1.9% (+58%)
- p-value: 0.001 (통계적 유의)

✅ 변형 B 채택
```
