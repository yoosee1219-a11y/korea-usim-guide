# PM: Project Planning Skill

프로젝트 매니저 관점에서 프로젝트를 계획하고 관리합니다.

## 기능:

1. **프로젝트 분석**
   - 요구사항 분석
   - 기술 스택 선정
   - 아키텍처 설계
   - 일정 수립

2. **태스크 분해**
   - Epic → Story → Task로 분해
   - 우선순위 설정
   - 의존성 파악
   - 리소스 할당

3. **진행 상황 추적**
   - 완료율 계산
   - 병목 구간 파악
   - 리스크 관리
   - 일정 준수율 체크

4. **보고서 생성**
   - 주간 진행 보고서
   - 마일스톤 달성 현황
   - 이슈 트래킹
   - 다음 단계 제안

## 사용 예:

```
"새로운 기능 개발 계획 세워줘"
→ Epic 분해 → Stories 생성 → Tasks 할당 → 일정 수립
```

## 출력:

- PROJECT_PLAN.md (프로젝트 계획서)
- TASKS.md (태스크 리스트)
- TIMELINE.md (타임라인)
- RISK_ANALYSIS.md (리스크 분석)
