# Quick Fix: Instant Problem Solving Skill

빠른 문제 해결 및 핫픽스

## 기능:

1. **에러 즉시 수정**
   - 빌드 에러 자동 해결
   - 타입 에러 자동 수정
   - Import 경로 자동 수정
   - 의존성 충돌 해결

2. **버그 핫픽스**
   - 버그 재현
   - 원인 파악
   - 수정 적용
   - 테스트 실행
   - 긴급 배포

3. **성능 즉시 개선**
   - 느린 컴포넌트 최적화
   - API 응답 속도 개선
   - 번들 사이즈 즉시 감소
   - 메모리 사용량 감소

4. **보안 즉시 패치**
   - 취약점 패치
   - 의존성 업데이트
   - 시크릿 제거
   - 권한 수정

## 사용 예:

```
"빌드가 안돼"
→ 에러 로그 분석 → 원인 파악 → 자동 수정 → 빌드 성공

"페이지가 너무 느려"
→ 성능 프로파일링 → 병목 찾기 → 최적화 → 성능 개선

"보안 경고가 떠"
→ 취약점 스캔 → 영향 분석 → 패치 적용 → 검증
```

## 실행 속도:

- 에러 분석: 10초
- 수정 적용: 30초
- 검증 완료: 1분
- **총 소요 시간: ~2분**

## 출력:

- QUICK_FIX_REPORT.md
- 수정된 파일 목록
- 테스트 결과
- 배포 여부
